A Stream in Java is not a data structure.
It is a pipeline for processing data.

A stream is a way to move data through a series of operations 
    (filtering, transforming, combining) without actually storing or 
    modifying the original data.

Has 3 parts
Source ->           Intermediate Operations -> Terminal Operations
(names.stream())   (filter, map, sorted)       (collect, forEach, reduce)
                    (These do nothing yet)    (This triggers the actual processing)

STREAMS ARE LAZY!!
THEY ONLY PROCESS WHEN A RESULT IS NEEDED

names.stream()
     .filter(n -> {
         System.out.println("Filtering...");
         return n.startsWith("A");
     });

Nothing prints because no terminal operation was called
